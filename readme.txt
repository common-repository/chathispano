Integra los servicios de la red de IRC & Chat de ChatHispano en tu WordPress. Inserta un Webchat en tu Web para chatear y conocer a la gente.

== Description ==

English:
The plugin allows a webmaster to insert webchat to chat in the ChatHispano Chat/IRC Network.

Spanish:
Este plugin permite a un webmaster insertar un webchat para chatear en la red de IRC/Chat de ChatHispano.

== Installation ==

Para instalar el plugin.

1. Descarga el plugin, descomprime el archivo y súbelo a la carpeta `/wp-content/plugins/` o instala el plugin a través de la pantalla de plugins de WordPress.
2. Activa el plugin a través del menú 'Plugins' en WordPress.
3. Configura las diferentes opciones en el menú 'ChatHispano' en WordPress.

Para insertar el WebChat en una página, hay que insertar el siguiente código:

`[chathispano_webchat]`


Puedes especificar canal para una pagina especifica en vez de usar el canal por defecto configurado con:

`[chathispano_webchat chan=#Madrid]`


Puedes especificar nick para una página específica en vez de usar el nick por defecto configurado con:

`[chathispano_webchat nick=ChatHispano_?]`
